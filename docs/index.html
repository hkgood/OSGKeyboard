开箱可用，只要三步

不用订阅、不用注册账号。装好权限，说第一句就行。

装好并授权

iOS：设置 → 键盘 → 开启完全访问。Mac：麦克风 + 辅助功能。完成时键盘列表能看到 OSG，或菜单栏出现图标。

先本地，Key 可选

默认本地识别即可开口；要 AI 润色再填自己的 API Key。不配 Key 也能落字。

开口即写

iOS 切到 OSGKeyboard 点麦克风；Mac 按住 Option。文字出现在当前光标处。
