安装并授权

iOS 添加键盘并开启完全访问；Mac 授予麦克风与辅助功能权限。

选引擎与润色

本地识别 + 内置润色零配置可用；也可填入自己的 API Key 接入任意兼容模型。

开口说话

切换到 OSGKeyboard 键盘，或按住 Option 键。润色好的文字自动插入光标处。
